# Next Demo

Basic Next.js demo app.

## Inspration

[Coursera's Coursera Introduction to Next.js](https://www.coursera.org/learn/introduction-to-next-js) served as starting point.

## Notes

[Next.js installation](https://nextjs.org/docs/app/getting-started/installation)

```bash
npx create-next-app@latest next-demo-app --yes
cd next-demo-app
npm run dev
```
